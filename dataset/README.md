# Dataset Notes

Place YOLO-format images and labels in this structure:

```text
dataset/
├── images/
│   ├── train/
│   ├── val/
│   └── test/
└── labels/
    ├── train/
    ├── val/
    └── test/
```

Recommended capture checklist:

- front and back side of notes
- both flat and folded notes
- bright and dim lighting
- cluttered and clean backgrounds
- single and multiple currency items in one frame
- Raspberry Pi camera captures
- motion blur from handheld camera movement

Make sure file names match:

- `images/train/img001.jpg`
- `labels/train/img001.txt`
